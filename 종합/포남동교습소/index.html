포남동 교습소
지문 속 주장과 근거 관계를 명확히 구분하도록 훈련함으로써 논리적 사고력을 강화한다. 예를 들어, 같은 개념을 다루는 두 교재의 설명 방식을 나란히 놓고 비교하면서 ‘어떤 표현이 더 이해하기 쉬웠는지’, ‘다른 점은 무엇인지’를 질문으로 기록하면 사고력이 자연스럽게 확장된다. 이러한 이해는 실수 감소로 이어지고, 더 복잡한 문제 해결의 기반이 된다. 포남동 교습소은 특히 ‘원주각’과 같은 추상적 개념을 이해하기 위해서는 이론적인 설명에 머무르지 않고 실제 문제 적용 과정을 반복함으로써 개념의 내재화를 촉진한다. 이때 ‘이 문제 맞았어요?’라는 예/아니오 확인 대신 ‘왜 이렇게 생각했는지’를 논리적으로 설명하도록 유도하면, 습관적으로 외우던 학습 방식에서 벗어나 사고의 구조를 스스로 점검하게 된다. 포남동 교습소은 또한, 개념 연결망 형성 여부를 정기적으로 확인함으로써 학습 내용 간 연계성을 강화하고, 통합적 사고력을 배양한다. 또한 요소별 개념 누락을 탐지하는 활동을 정기적으로 시행함으로써, 학생이 놓친 부분을 신속히 파악하고 보완할 수 있다.
